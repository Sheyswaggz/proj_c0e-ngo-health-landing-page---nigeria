# NGO Health Landing Page - Nigeria

A mobile-first, low-bandwidth optimized landing page for a Nigerian health NGO, designed to provide accessible health information and resources across diverse connectivity conditions.

## Mission Context

This project serves a Nigerian health NGO dedicated to improving healthcare access and outcomes in underserved communities. The landing page is optimized for:

- **Mobile-first experience** - Most users access via smartphones
- **Low-bandwidth environments** - Minimal asset sizes and efficient loading
- **Accessibility** - WCAG 2.1 AA compliance for inclusive access
- **Local context** - Nigerian-themed design and culturally relevant content

## Technology Stack

- **HTML5** - Semantic markup for accessibility and SEO
- **Tailwind CSS** (via CDN) - Utility-first styling without build complexity
- **Vanilla JavaScript** - Lightweight interactivity with zero dependencies
- **Python HTTP Server** - Simple local development server

### Why This Stack?

- **Zero build process** - Immediate development without tooling overhead
- **CDN delivery** - Fast, cached asset delivery
- **Minimal dependencies** - Reduced maintenance burden
- **Universal compatibility** - Works across all modern browsers
